# <golfers>

## サイト概要
・プレーしたゴルフ場のレビューサイト

### サイトテーマ
・ゴルフをまだ始めたばかりやコース選びに迷っている人向けに、コースを選びやすくなるようなサイト

### テーマを選んだ理由
・どんなコースが良いのかなど悩んでしまうことが多かったので、選びやすくなるようにしたかったため

### ターゲットユーザ
・ゴルフ初心者や若者向け

### 主な利用シーン
・ゴルフ場を選ぶ際やプレーした後
